## What `instructions` actually changed

1. **Faster routing on host vs container questions** — N3's −57% is the clearest signal. The "per-host vs per-container" line in `instructions` shortcut a tool-call round-trip.
2. **`search` still used when it's right** — W1 confirms instructions did NOT make the model duck the escape hatch. The "prefer specific tool over `search`" phrasing didn't over-steer.
3. **No verbosity blowup** — answers are equal or tighter; no measurable cost from the +497-char `instructions` payload.

## What instructions did NOT fix

- **I2 still wrong** — model answered with containers for "what's hammering the box?" The current instructions wording explains the host/container distinction but doesn't address "the box" ambiguity. Wording sharpening is a low-cost follow-up if we want to push further; not a rollback trigger.
